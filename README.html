Aegis NFC 시스템 가이드

Smart RFID Access Control System Developer Guide

1. 시스템 개요

Aegis NFC는 ESP8266 NodeMCU 보드와 PN532 RFID 모듈, 0.96인치 OLED, 그리고 패시브 부저를 결합한 하드웨어 디바이스와 AWS EC2 상의 웹 대시보드 서비스가 상호작용하는 실시간 출입 통제 관리 시스템입니다.

2. 하드웨어 결선 사양

OLED 디스플레이와 PN532 모듈은 I2C 버스를 병렬로 안전하게 공유하여 작동합니다.

연결 장치 장치 핀 NodeMCU 핀 설명
PN532 RFID
(I2C 모드: 1번 ON, 2번 OFF)		 VCC 3V3 (또는 VU) 전원 입력
GND GND 접지
SDA D5 (GPIO 14) I2C 데이터 공유선
SCL D6 (GPIO 12) I2C 클럭 공유선
0.96" OLED
(주소: 0x3C)		 SDA D5 (GPIO 14) I2C 데이터 공유선
SCL D6 (GPIO 12) I2C 클럭 공유선
패시브 부저 + (긴 다리) D7 (GPIO 13) 주파수 신호 입력 (저항 100~220Ω 직렬 연결)
- (짧은 다리) GND 접지

3. 데이터베이스 스키마 (MySQL / MariaDB)

AWS EC2 내부에 구축된 aegis_nfc 데이터베이스 테이블 DDL 정의서입니다. 데이터베이스 전용 사용자는 aegis_user (비밀번호: aegis_password) 로 연결하도록 안전하게 구성되었습니다. 

-- 1. 사용자 테이블
CREATE TABLE IF NOT EXISTS users (
  uid VARCHAR(20) PRIMARY KEY,
  name VARCHAR(50) NOT NULL,
  dept VARCHAR(50) NOT NULL,
  phone VARCHAR(20) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 2. 실시간 출입 이력 로그 테이블
CREATE TABLE IF NOT EXISTS logs (
  id INT AUTO_INCREMENT PRIMARY KEY,
  uid VARCHAR(20) NOT NULL,
  name VARCHAR(50) NOT NULL,
  dept VARCHAR(50) NOT NULL,
  status VARCHAR(20) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

4. 백엔드 PHP API 명세서

모든 API의 루트 경로는 http://54.116.172.123/api/ 입니다. (모든 시각은 한국 표준시 KST 기준으로 자동 보정됩니다.)

POST /api/access.php
RFID 카드 접촉 시 데이터 확인 및 로그 적재

Request Body (JSON):

{ "uid": "84 AC A8 A0" }

Response (JSON - 등록 카드인 경우):

{
  "status": "APPROVED",
  "name": "김준석",
  "dept": "연구소 개발팀",
  "uid": "84 AC A8 A0"
}

Response (JSON - 미등록 카드인 경우):

{
  "status": "DENIED",
  "name": "Unknown",
  "dept": "미등록 방문객",
  "uid": "99 AA BB CC"
}
GET /api/logs.php
대시보드 실시간 렌더링용 최근 20개 출입 로그 조회

Response (JSON):

[
  {
    "id": 15,
    "uid": "84 AC A8 A0",
    "name": "김준석",
    "dept": "연구소 개발팀",
    "status": "APPROVED",
    "time": "16:45:12",
    "created_at": "2026-06-24 16:45:12"
  }
]
GET /api/stats.php
상단 수치 및 부서별 통계 집계

Response (JSON):

{
  "total": 12,
  "approved": 10,
  "blocked": 2,
  "dept_stats": [
    { "dept": "연구소 개발팀", "count": 8 },
    { "dept": "영업 기획팀", "count": 2 }
  ],
  "time_stats": [
    { "hour": 16, "count": 12 }
  ]
}
POST /api/register.php
미등록 카드 팝업 모달을 통한 신규 사용자 등록

Request Body (JSON):

{
  "uid": "99 AA BB CC",
  "name": "홍길동",
  "dept": "네트워크 보안팀",
  "phone": "010-9999-8888"
}

Response (JSON):

{
  "status": "SUCCESS",
  "message": "User registered successfully."
}
GET /api/download_csv.php
데이터베이스 전체 로그 CSV 파일 엑셀 다운로드 (UTF-8 BOM 탑재)

5. 로컬 아두이노 스케치 가이드

지금까지 로컬에 작성된 네 가지 스케치 빌드 폴더들입니다.

  • OledAnimations: 내장 OLED 디스플레이를 활용한 5종(Starfield, 3D Cube 등) 렌더링 루프 예제
  • RfidTest: I2C 버스를 공유하는 PN532 리더기와 부저 테스트용 소리 나는 카드 리딩 코드
  • WifiTime: JUNSEOKIM 와이파이 연결 및 NTP를 이용해 OLED에 KST 시계 시각화
  • RfidAws: (최종) 와이파이 자동 연결 및 실시간 AWS access.php API 연동, OLED 이름/소속 표시, 부팅/성공/거절 사운드 개별 피드백이 완성된 실시간 연동 펌웨어

아두이노 CLI를 사용해 보드에 컴파일 및 업로드할 때는 아래 명령어를 사용해 세션 환경변수(PATH)를 포함해 실행합니다:
$env:PATH += ";C:\Program Files\Arduino CLI"
arduino-cli compile --fqbn esp8266:esp8266:nodemcuv2 <폴더명>
arduino-cli upload -p COM3 --fqbn esp8266:esp8266:nodemcuv2 <폴더명>
